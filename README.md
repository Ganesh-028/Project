# AI Resume Builder

A modern, functional AI-powered resume builder built with React and Vite. This application allows users to create professional, ATS-optimized resumes efficiently.

## Features

- **Step-by-Step Editor**: Fill out your details (Personal, Experience, Education, Skills, Certifications, Projects) in a structured, intuitive interface.
- **AI Integration**:
  - Automatically generate compelling professional summaries.
  - Enhance your experience bullet points with AI-powered suggestions.
- **Real-time Preview**: See your resume update instantly as you type.
- **Multiple Premium Templates**: Switch between Professional, Modern, and Creative layouts seamlessly.
- **Typography Customization**: Choose your preferred font family and base font size to tailor the look of your resume.
- **ATS Score Calculation**: Get instant feedback on how well your resume might perform in Applicant Tracking Systems based on the provided content.
- **PDF Export**: Download your finished resume as a high-quality PDF.
- **Theming**: Toggle between light and dark modes for a comfortable editing experience.

## Getting Started

### Prerequisites
- Node.js installed on your machine.

### Installation

1. Clone the repository or download the source code.
2. Install the dependencies:
   ```bash
   npm install
   ```

### Development

To start the local development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Exporting PDF
Click the "Download PDF" button in the sidebar or the "PDF" button in the Live Preview panel to generate and print your resume.

## Technologies Used

- React 18
- Vite
- Lucide React (Icons)
- React-to-Print (PDF generation)
- Vanilla CSS (Premium glassmorphic design)
